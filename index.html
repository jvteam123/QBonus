    Formula
    Quality % = Fix ÷ (Fix + Refix + Warnings) × 100
    Fix = tasks where tech is FIX1_ID … FIX5_ID
    Refix = tasks with refix label (e.g. i) in RV columns
    Warnings = tasks with warn labels in warn columns

    Standard formula
    Payout = TotalPoints × Multiplier × (BonusEarned% ÷ 100)
    BonusEarned% is looked up from the Quality % Modifier table. Multiplier is set in the Bonus Calculator panel.
