    7 i3 Penalty Structure (feedback by i3QA Tech — deducted from QC Tech)
    Who gets deducted?
    The QC Tech (QC_ID) on each missed row receives the DR-level point deduction from their total fixpoints.
    The i3QA Tech is not deducted by this penalty — they caused the feedback.
    What triggers it?
    Egregious rows = I3QA_CAT (or CATEGORY) is 1, 2, or 3
    Count is summed per QC Tech, per project cut-off
    Highest matching DR level applies (not cumulative)
    Egregious Categories: Cat 1, 2, 3  ·  Deductions per Egregious Count (project per cut-off)
    DR Level Egregious Count Range Action Fixpoints Deduction
    DR130 CountsCoaching Memo-30 pts
    DR250 CountsCoaching Memo-50 pts
    DR370 CountsWritten Warning and Notice to Explain-100 pts
    DR4100 CountsWritten Warning and Notice to Explain-200 pts
    DR5200 CountsCoaching Memo with Class B-1 Offenses-500 pts
    • Class B-1: These are moderate offenses with a 4-offense progression leading to termination
    8 i3QA Penalty Structure (feedback by RV's and US — deducted from i3QA Tech)
    Who gets deducted?
    The i3QA Tech (I3QA_ID) on each missed egregious row receives the DRQ-level deduction.
    Rows where FIXED1? = "N" (not yet fixed) or AFP1_STAT = "AA" are counted toward the DRQ threshold.
    DRQ Trigger Conditions
    An egregious row is counted toward DRQ if FIXED1? = "N" (unfixed) or AFP1_STAT = "AA" (RQA-approved).
    Rows that meet neither condition are excluded from the DRQ count.
    Egregious Categories: Cat 1, 2, 3  ·  DRQ count = egregious rows where FIXED1? = "N" or AFP1_STAT = "AA"  ·  per project cut-off
    DRQ Level Egregious Count Range Action Fixpoints Deduction
    DRQ110 CountsCoaching Memo-30 pts
    DRQ220 CountsCoaching Memo-50 pts
    DRQ350 CountsWritten Warning and Notice to Explain-100 pts
    DRQ4100 CountsWritten Warning and Notice to Explain-200 pts
    DRQ5200 CountsCoaching Memo with Class B-1 Offenses-500 pts
    • Class B-1: These are moderate offenses with a 4-offense progression leading to termination
    9 End-to-End Final Points Formula — how every number is computed
    QC Tech Final Points
    + (Initial rows tagged × 0.125)
    − (QC miss rows × 0.125)
    = Base fixpoints
    − i3 DR Penalty (if egregious ≥ threshold)
    = Final Points
    i3 DR Penalty uses ALL egregious rows (Cat 1,2,3) — no FIXED1 filter.
    QC miss deduction (0.125/row) is separate from the DR-level penalty.
    i3QA Tech Final Points
    + (i3QA task rows × 1/12)
    + (QC miss rows × 0.125) ← QC Transfer
    = Base fixpoints
    − i3QA DRQ Penalty (if FIXED1=N or AFP1=AA egregious ≥ threshold)
    = Final Points
    DRQ Penalty counts egregious rows where FIXED1? = "N" or AFP1_STAT = "AA".
    Rows meeting neither condition are excluded from the DRQ threshold count.
    Complete Calculation Flow
    ① Initial Tagging
    QC_ID rows × 0.125
    ② QC Miss Penalty
    −0.125 per miss row
    from QC Tech
    ③ QC Transfer
    +0.125 per miss row
    to i3QA Tech
    ④ DR / DRQ Level
    DR = all egregious
    DRQ = FIXED1=N or AFP1=AA
    ⑤ Final Points
    Base + Penalties
    = Payout basis
    DR (i3 Penalty) — attributed to QC Tech
    Threshold: 30→DR1(−30), 50→DR2(−50), 70→DR3(−100), 100→DR4(−200), 200→DR5(−500)
    Basis: all egregious (Cat 1,2,3) rows regardless of FIXED1
    DRQ (i3QA Penalty) — attributed to i3QA Tech
    Threshold: 10→DRQ1(−30), 20→DRQ2(−50), 50→DRQ3(−100), 100→DRQ4(−200), 200→DRQ5(−500)
    Basis: only egregious rows where FIXED1? = "N" or AFP1_STAT = "AA"
    All values editable under ⚙ Settings & Roster · Changes apply immediately to all calculations
